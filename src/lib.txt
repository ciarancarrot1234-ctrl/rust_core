//! # Liquidity Hunt
//!
//! High-frequency trading system for Binance Futures with real-time market data processing,
//! event-driven architecture, and comprehensive trade analysis.
//!
//! ## Architecture
//!
//! The system is organized into layers:
//!
//! - **Core**: Foundational types, configuration, logging, and event bus
//! - **Layer 1**: Data connectors (WebSocket, REST, User Data Stream, Time Sync)
//! - **Layer 2**: Data processing pipeline (Parser, OrderBook, Synchronizer)
//! - **Layer 3**: Trade aggregators and event generators (Delta, VWAP, Volume, Patterns)
//!
//! ## Example
//!
//! ```rust,no_run
//! use liquidity_hunt::core::setup_logging;
//! use liquidity_hunt::layer1::WebSocketClient;
//! use liquidity_hunt::layer2::RustPipeline;
//!
//! // Setup logging
//! setup_logging(Some("INFO"), None, Some(true), Some(true));
//!
//! // Create and start pipeline
//! let pipeline = RustPipeline::new("BTCUSDT", "differential");
//! pipeline.initialize().unwrap();
//! pipeline.start("depth@100ms").unwrap();
//! ```
